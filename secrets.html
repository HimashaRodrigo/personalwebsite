What is secrets app?

This is actually a simple web application however the focus of this project was to create a secure web application for users to post their secrets and see others secrets anonymously means people can post their own secrets and can enjoy seeing others' secrets Hiding their identity.


Used technologies
  • HTML, CSS, bootstrap, ejs, nodejs, express, mongoDB.

How to use the app?
  • Firstly you need to register or login in to the web app. When registering or loging you can use your email and password and also you can sign up using a Google or Facebook login.



  • After the registration is done you can access the secrets page then you can submit your secret with others.



  • The special feature is that you can visit the secret page to view other people's secrets without registering in the app, but you can't submit your own secret if you don't registered.
